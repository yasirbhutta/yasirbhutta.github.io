## 4.6 Polynomial Multiplication

**Example:** Multiply 2x^2 Ã— 3x

```matlab
x = [2 0 0];
y = [3 0];

z = conv(x,y);
disp(z);
```

**Output:**
```
6     0     0     0
```

The polynomial is **6x^3**
