### **Day 4: Technical Analysis**  
**ğŸ“Œ Task:** Predict trends with moving averages  
- **Code:**  

### **Day 5: Volatility Analysis**

- **Discussion:**  
  - High volatility â†’ High risk/reward (Tesla vs. S&P 500).
